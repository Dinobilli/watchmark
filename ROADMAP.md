# Roadmap

## v0.1 - Public MVP

- Local dashboard.
- URL check API.
- Baseline storage.
- Meaningful text-change detection.
- Diff view with added and removed meaningful lines.
- Local summaries.
- Private URL guard.
- CI, tests, and maintainer docs.

## v0.2 - Durable Monitoring

- SQLite persistence.
- Scheduled checks.
- Watch history.
- Exportable fixtures for public pages.
- More Korean public-sector and shopping-page examples.

## v0.3 - Alerts

- Email and webhook notifications.
- Slack and Discord integrations.
- Alert grouping and quiet hours.
- Richer diff filters for noisy dynamic sections.

## v0.4 - Hosted-Ready Core

- Team workspaces.
- Watch limits.
- Retention policies.
- Optional OpenAI-powered summaries and evaluations.
- Admin-facing maintainer automation for issue triage and release notes.
